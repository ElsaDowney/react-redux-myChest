请按以下步骤运行：
    1. 编辑器安装editorconfig插件
    2. clone
    3. cnpm i
    4. npm run webpack
    5. npm start
    6. 测试:npm test
    7. 提交代码前运行npm run eslint检查代码格式
注意:本项目使用js框架为mocha,与jasmine的写法有些区别
